# Mini LinkedIn Community Platform 👥

A mini LinkedIn-like web application where users can register/login, create posts, view a feed, and visit user profiles.

---

## 🛠️ Stack Used

### 🧑‍💻 Frontend
- **React** (Vite)
- **Tailwind CSS**
- **React Router DOM**

### 🧑‍🍳 Backend
- **Node.js**
- **Express.js**
- **MongoDB** (via **Mongoose**)
- **JWT** (for authentication)
- **dotenv**
